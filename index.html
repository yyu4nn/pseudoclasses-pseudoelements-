PSEUDO-CLASSES & PSEUDO-ELEMENTS CSS


¿Que es una Pseudo-class ?


Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado. Por ejemplo, :hover aplicará un estilo cuando el usuario haga hover sobre el elemento especificado por el selector.

Al igual tambien existen Sub-classes

¿Que es un Pseudo-element ?


Al igual que las pseudo-classes, los pseudo-elementos se añaden a los selectores, pero en cambio, no describen un estado especial sino que, permiten añadir estilos a una parte concreta del documento. Por ejemplo, el pseudoelemento ::first-line selecciona solo la primera línea del elemento especificado por el selector.

Algunas de las que funcionan


:any-link

Representa a un elemento que actúa como el ancla origen de un hipervínculo independientemente de si ha sido visitado, es decir, coincide con cualquier elemento "a", "area" o "link" con un atributo href. Por lo tanto, coincide con todos los elementos que coincidan con :link o :visited.

:current

Es una pseudoclase temporal dimensional que representa un elemento o el antepasado de un elemento que se está mostrando actualmente. Por ejemplo, esta pseudoclase se puede utilizar para representar un vídeo que WebVTT muestra con subtítulos.

:has

Representa un elemento si alguno de los selectores relativos que se pasan como argumento coincide con al menos un elemento cuando se ancla contra este elemento. Esta pseudoclase presenta una forma de seleccionar un elemento padre o un elemento hermano anterior con respecto a un elemento de referencia tomando una lista de selectores relativos como argumento.

:future

Es una pseudoclase temporal que coincidirá con cualquier elemento que aparezca completamente después de un elemento que coincida con :current. Por ejemplo, en un video con subtítulos que WebVTT está mostrando.




:focus-visible

Se aplica mientras un elemento coincide con la pseudo-clase :focus y el UA (User Agent) determina a través de heurísticas que el foco debe hacerse evidente en el elemento. (Muchos navegadores muestran un "anillo de enfoque" por defecto en este caso).

Coincide con un elemento si el elemento o cualquiera de sus descendientes están enfocados. En otras palabras, representa un elemento que coincide con la pseudoclase :focus o tiene un descendiente que coincide con . (Esto incluye descendientes en árboles de sombra).:focus

:past

Es una pseudoclase temporal dimensional que coincidirá con cualquier elemento que aparezca completamente antes de un elemento que coincida con :current. Por ejemplo, en un video con subtítulos que WebVTT está mostrando.

:placeholder-shown

Representa cualquier elemento "input" o "textarea" que actualmente muestra texto de marcador de posición.

PSEUDO-ELEMENTS


::after

Es el último elemento secundario del elemento seleccionado. A menudo se usa para agregar contenido cosmético a un elemento con la propiedad content. Está en línea por defecto.

Here is some plain old boring text.

Here is some normal text that is neither boring nor exciting.

Contributing to MDN is easy and fun.

::before

Es el primer elemento secundario del elemento seleccionado. A menudo se usa para agregar contenido cosmético a un elemento con la propiedad content. Está en línea por defecto.

Notice where the orange box is.

::file-selector-button

representa el botón de un "input" de type="file".



::first-letter

Aplica estilos a la primera letra de la primera línea de un elemento de nivel de bloque, pero solo cuando no está precedido por otro contenido (como imágenes o tablas en línea).

::first-line

Aplica estilos a la primera línea de un elemento de nivel de bloque.

Styles will only be applied to the first line of this paragraph. After that, all text will be styled like normal. See what I mean?

::grammar-error

Representa un segmento de texto que el agente de usuario ha marcado como gramaticalmente incorrecto.

My friends is coming to the party tonight.

::highlight

Aplica estilos a un resaltado personalizado.

Un resaltado personalizado es una colección de objetos Range y se registra en una página web mediante HighlightRegistry.

CSS Custom Highlight API rainbow



::marker

Selecciona el cuadro de marcador de un elemento de lista, que normalmente contiene una viñeta o un número. Funciona en cualquier elemento o pseudoelemento establecido para mostrar: list-item, como los elementos "li" y "summary".

  • Peaches
  • Apples
  • Plums

::place-holder

Representa el texto del marcador de posición en un elemento "input" o "textarea".

::selection

Aplica estilos a la parte de un documento que el usuario ha resaltado (como hacer clic y arrastrar el ratón por el texto).

This text has special styles when you highlight it.

Also try selecting text in this paragraph.

representa un segmento de texto que el agente de usuario ha marcado como mal escrito.
